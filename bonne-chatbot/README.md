# Bonne Chatbot — Setup Guide

## What's in this folder

```
bonne-chatbot/
  data/
    bone_metastases_text.txt   ← PDF text already extracted for you
    vectorstore.json           ← generated by ingest.js (run step 2)
  scripts/
    ingest.js                  ← embeds your docs, creates vectorstore.json
  backend/
    chat.js                    ← add this router to your existing server.js
  frontend/
    ChatPanel.js               ← add this component to your Next.js pages
  package.json
  env_additions.txt            ← copy these lines into backend/.env
```

## Step 2 — Install deps and run the ingestion script

Open PowerShell in `E:\boneguard-app\bonne-chatbot\` and run:

```powershell
npm install
node scripts/ingest.js
```

This will:
- Read bone_metastases_text.txt
- Split it into chunks
- Call Gemini embedding API for each chunk
- Save data/vectorstore.json

Takes about 30-60 seconds. You'll see progress printed.

Expected output:
```
Found 1 document(s): bone_metastases_text.txt
  > bone_metastases_text.txt: 14 chunks
Total chunks to embed: 14
Calling Gemini text-embedding-004...
  Embedding chunk 1/14... done
  ...
  Embedding chunk 14/14... done
Vector store saved to: data/vectorstore.json
Total chunks embedded: 14
```

---

## Step 3 — Add chat router to backend/server.js

Open `E:\boneguard-app\backend\server.js` and add these 2 lines
near the top with the other requires:

```js
const chatRouter = require("../bonne-chatbot/backend/chat");
```

And add this line after app.use(cors(...)):

```js
app.use("/api", chatRouter);
```

So it looks like:

```js
// ... existing code ...
const chatRouter = require("../bonne-chatbot/backend/chat");

// ... existing middleware ...
app.use(cors({ ... }));
app.use(express.json());
app.use("/api", chatRouter);   // ← add this line
// ... rest of existing routes ...
```

## Step 4 — Add ChatPanel to your Next.js frontend

Copy ChatPanel.js to:
  E:\boneguard-app\frontend\components\ChatPanel.js

Then open `E:\boneguard-app\frontend\pages\index.js`
and add these two lines:

At the top with other imports:
```js
import ChatPanel from "../components/ChatPanel";
```

At the bottom inside the main return, just before the closing </div>:
```jsx
<ChatPanel />
```

It should look like:
```jsx
      </footer>
      <ChatPanel />   {/* ← add this */}
    </div>
  );
}
```

## To add more PDFs later

1. Copy your PDF text into a new .txt file in bonne-chatbot/data/
   (Use any free PDF-to-text converter, or ask me to extract it)
2. Run node scripts/ingest.js again
3. Restart the backend server

The vector store rebuilds automatically with all .txt files in /data/
